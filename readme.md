## üöÄ Core Projects

### üü¢ Echoes of Mind (Founder)

A behavioral intelligence platform that goes beyond mood tracking.

**Problem:**
Most mental health apps track emotions without understanding patterns.

**Solution:**
Build a signal-detection engine that identifies behavioral shifts across time windows (24h, 7d, 30d) and triggers insights only when statistically meaningful changes occur.

**Architecture Highlights:**

* React Native (performance-focused builds)
* NestJS modular backend
* Prisma schema-first modeling
* MySQL with structured indexing
* Redis for caching & rate limiting
* Behavioral detectors (frequency spikes, volatility increase, night clustering, intensity shifts)
* Push notification system tied strictly to behavioral signals
* Device token management with lifecycle handling
* Cron-based execution layer
* AI-powered NLP pipeline (sentiment, entity extraction, trigger analysis)

Design philosophy:
No noise. No artificial engagement loops. Insights only when justified by data.
